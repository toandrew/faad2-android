##
Convert aac to wav files

## HOW-TO-USE
1. Set `PATH` to `ndk-build`
2. Clone this repository
```
git clone https://github.com/toandrew/faad2-android.git --recursive
```
3. Run build shell
```
cd faad2-android
./android_build.sh
```

4. open 'samples' project
